Getting Started

Set up the Couchbase Query Analyzer with AI-powered analysis

1. Get the Tool

You can run the tool using Docker or by downloading an executable.

Pull from Docker Hub 
docker pull fujioturner/couchbase-query-analyzer:latest
docker run -p 5000:5000 fujioturner/couchbase-query-analyzer:latest

After running, open http://localhost:5000 in your browser.

Download for macOS
  1. Download the cb_query_analyzer_macos executable.
  2. Open Terminal and navigate to the download location.
  3. Make it executable: chmod +x cb_query_analyzer_macos
  4. Run it: ./cb_query_analyzer_macos
  5. Note: If you see a security warning, go to System Settings > Privacy & Security and allow the app to run.

After running, open http://localhost:5000 in your browser.

Download for Windows
  1. Download the cb_query_analyzer.exe file.
  2. Double-click to run.
  3. Note: If you see a "Windows protected your PC" warning, click "More info" and then "Run anyway".

After running, open http://localhost:5000 in your browser.

2. Connect to Couchbase

You need to connect the tool to a Couchbase cluster to save analysis data and user preferences.

Start a Free Trial or Sign In

Setup Steps

  1. Create Bucket: Create a bucket named cb_tools (Quota: 256 MB).
  2. Create Scope/Collection: In cb_tools, create scope query and collection analyzer.
    (Ensure _default scope/collection exists for preferences)
  3. Create Index:

    Run the following query in your Query Workbench:

    CREATE INDEX `analysis_old_table_v2` 
ON `cb_tools`.`query`.`analyzer`(
    `createdAt` DESC INCLUDE MISSING,
    `metadata`,
    `status`,
    `prompt`,
    `provider`,
    `sourceCluster`,
    `parseJson`.`filters`
) 
WHERE (`docType` = "ai_analysis")
  4. Create User: Create Database Credential
    Roles: Read/Write access to cb_tools bucket (and all scopes/collections)
  5. Whitelist IP: Add your IP Address to Allowed List
  6. Connection Details:
    URL: Your Capella Connection String (e.g., couchbases://...)
    Username: Your DB Credential
    Password: Your Password

3. Configure the Tool

Once the tool is running and your Couchbase user is ready:

  1. Open your browser and go to http://localhost:5000.
  2. Click the Settings (Gear Icon) in the upper right corner.
  3. Enter your Cluster URL, Username, and Password.
  4. Click Test Connection.
    • If successful, you will see a confirmation message.
  5. Click Save Settings.
